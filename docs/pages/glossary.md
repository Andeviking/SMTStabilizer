\page page_glossary Glossary

## Context Hash

A hash value representing structural context influence around a node.

## Processing Slice

A subset of node indices currently involved in propagation/refinement.

## Symbol Bucket

A symbol group that currently collides under the active hash strategy.

## Symmetry Breaking Perturbation

A controlled tie-breaking technique used to separate otherwise identical symbol groups.

## Sort Propagation

A stage that injects sort/datatype distinctions into propagation and ordering.
